# 家庭能耗管家 - 水电气全能版

一个用于记录、分析和预测家庭水、电、气三种能源消耗的现代化Web应用，支持智能OCR识别账单、季节性趋势预测、多视图数据分析和本地数据同步。

## ✨ 主要功能

### 📊 数据管理
- **三合一管理**: 电费、水费、燃气费统一记录
- **手动录入**: 按月录入用量和费用
- **智能OCR**: 批量上传账单图片，自动识别数据（支持支付宝/微信/国网/燃气公司账单）
- **数据导入导出**: 支持JSON、文本文件导入；导出为JSON格式备份
- **本地同步**: 从`data`文件夹自动同步数据，支持`data.json`统一格式和分文件格式

### 📈 智能分析
- **月度对比**: 多年度用量和费用对比，支持按年份筛选
- **阶梯累计分析**: 年度累计用量与阶梯计价突破点分析
- **阶梯预测**: 基于历史季节性趋势，预测进入下一阶梯的月份
- **同期对比**: 今年同期（1-N月）vs 去年同期用量和费用对比

### ⚙️ 阶梯计价
- **分类型设置**: 电、水、气分别设置阶梯标准
- **实时状态**: 显示当前阶梯和本年累计用量
- **智能预测**: 考虑历史季节性波动（夏季高峰、冬季低谷），预测阶梯突破时间

### 🎨 界面设计
- **三色主题**: 电费（紫）、水费（蓝）、燃气（橙）
- **响应式布局**: 支持桌面和移动端
- **关键指标卡片**: 年度费用、年度用量、当前阶梯、阶梯预测

## 📖 使用指南

### 1. 能源类型切换
顶部标签页切换三种能源：
- **电费**: 紫色主题，单位：度
- **水费**: 蓝色主题，单位：吨（双月计费）
- **燃气**: 橙色主题，单位：立方米（双月计费）

### 2. 阶梯计价设置
首次使用请根据当地政策设置阶梯标准：
- **电费**: 第一阶梯 0-3120度，第二阶梯 3121-4800度
- **水费**: 第一阶梯 0-180吨，第二阶梯 181-260吨
- **燃气**: 第一阶梯 0-300立方米，第二阶梯 301-600立方米

设置自动保存到本地存储。

### 3. 数据录入

#### 手动录入
在左侧"手动添加"区域填写日期、用量和费用。

#### OCR智能识别
1. 拖拽或点击上传账单图片
2. 系统自动识别用量、费用、日期
3. 核对后确认导入

#### 文件导入
支持格式：
- **JSON**: 新版统一格式或旧版单类型数组
- **文本**: 类似 `2023-01 用电量：123度 电费：¥234` 格式

#### 数据同步
- 数据自动保存到浏览器 LocalStorage
- 放置 `data.json` 到 `data/` 文件夹可自动同步

### 4. 数据分析

#### 月度对比视图
- 查看多年用量趋势
- 识别高峰/低谷月份
- 双月计费类型自动连线显示

#### 阶梯分析视图
- 查看累计用量曲线
- 阶梯突破点可视化
- 阶梯突破明细表（每年进入二阶、三阶的具体月份）

#### 阶梯预测
基于历史季节性趋势预测：
- 分析历史年份的月度分布比例
- 预测今年剩余月份用量
- 估算进入下一阶梯的时间

**示例预测结果**：
```
当前：2026年2月，累计964度，第一阶梯
预测：8月进入第二阶梯，12月进入第三阶梯
```

## 🔧 数据格式

### 推荐格式：`data.json`（统一格式）
将所有数据合并到一个文件中：
```json
{
  "version": "2.0",
  "data": {
    "electricity": [
      {"date": "2024-01", "usage": 748, "cost": 380}
    ],
    "water": [
      {"date": "2024-01", "usage": 12.5, "cost": 45.6}
    ],
    "gas": [
      {"date": "2024-01", "usage": 45.3, "cost": 123.4}
    ]
  },
  "tierSettings": {
    "electricity": {"tier1": 3120, "tier2": 4800},
    "water": {"tier1": 180, "tier2": 260},
    "gas": {"tier1": 300, "tier2": 600}
  }
}
```

### 兼容格式：分文件（旧版）
系统也支持分开存储的文件格式（优先读取 `data.json`，不存在则读取分文件）：
- `data/electricity.json`: 电费数据数组
- `data/water.json`: 水费数据数组
- `data/gas.json`: 燃气数据数组
- `data/config.json`: 阶梯设置（可选）

**分文件格式示例**（`electricity.json`）：
```json
[
  {"date": "2024-01", "usage": 748, "cost": 380},
  {"date": "2024-02", "usage": 762, "cost": 392.7}
]
```

### 字段识别
系统自动识别以下字段：
- **用量**: `usage`, `kwh`, `amount`, `用电`, `水量`, `气量` 等
- **费用**: `cost`, `fee`, `price`, `费用`, `金额` 等
- **日期**: `date` (格式: YYYY-MM)

### 示例文件
- `data/data.json`: 统一格式完整数据（推荐）
- `data/sample-electricity.json`: 电费示例（分文件格式）
- `data/sample-water.json`: 水费示例（分文件格式）
- `data/sample-gas.json`: 燃气示例（分文件格式）
- `data/sample-config.json`: 阶梯设置示例
- `sample-data.txt`: 文本格式示例
